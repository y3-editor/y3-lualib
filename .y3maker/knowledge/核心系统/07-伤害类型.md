# Y3 伤害类型

游戏中的伤害类型定义。

## 📋 伤害类型 (DamageType)

| 类型 | 值 | 说明 |
|------|---|------|
| `physical` | 0 | 物理伤害 |
| `magic` | 1 | 法术伤害 |
| `real` | 2 | 真实伤害（无视护甲和魔抗） |
| `adaptive` | 3 | 自适应伤害（根据目标属性选择最优伤害类型） |

---

## 📋 伤害计算

### 物理伤害

物理伤害受目标的**护甲值**影响：

```
实际物理伤害 = 基础伤害 × (1 - 护甲减伤)
```

护甲减伤公式（参考）：
```
护甲减伤 = 护甲 / (护甲 + 固定值)
```

### 法术伤害

法术伤害受目标的**魔法抗性**影响：

```
实际法术伤害 = 基础伤害 × (1 - 魔抗减伤)
```

### 真实伤害

真实伤害**不受任何减伤效果影响**，直接扣除目标血量：

```
实际真实伤害 = 基础伤害
```

### 自适应伤害

自适应伤害会根据目标的护甲和魔抗，选择能造成更高伤害的类型：

```lua
if 目标护甲 > 目标魔抗 then
    -- 按法术伤害计算
else
    -- 按物理伤害计算
end
```

## 🔧 物编中配置伤害类型

### 技能伤害类型

在技能物编中设置 `ability_attribute` 字段：

| 值 | 类型 |
|---|------|
| 0 | 物理伤害 |
| 1 | 法术伤害 |
| 2 | 真实伤害 |

### 护盾可抵挡的伤害类型

在Buff物编中设置 `shield_type` 字段：

| 值 | 说明 |
|---|------|
| 0 | 只抵挡物理伤害 |
| 1 | 只抵挡法术伤害 |
| 2 | 抵挡所有伤害（通用护盾） |

## 🔧 Lua 使用示例

### 造成指定类型伤害

```lua
-- 造成物理伤害
local function deal_physical_damage(source, target, amount)
    target:damage({
        source = source,
        damage = amount,
        damage_type = 0  -- physical
    })
end

-- 造成法术伤害
local function deal_magic_damage(source, target, amount)
    target:damage({
        source = source,
        damage = amount,
        damage_type = 1  -- magic
    })
end

-- 造成真实伤害
local function deal_true_damage(source, target, amount)
    target:damage({
        source = source,
        damage = amount,
        damage_type = 2  -- real
    })
end
```

### 根据属性选择伤害类型

```lua
-- 自适应伤害（模拟实现）
local function deal_adaptive_damage(source, target, amount)
    local armor = target:get_attr('armor')
    local magic_resist = target:get_attr('magic_resist')
    
    local damage_type
    if armor > magic_resist then
        damage_type = 1  -- 用法术伤害
    else
        damage_type = 0  -- 用物理伤害
    end
    
    target:damage({
        source = source,
        damage = amount,
        damage_type = damage_type
    })
end
```

## ⚠️ 注意事项

1. **护盾优先级**：伤害会先被护盾吸收，剩余伤害才扣除血量
2. **伤害顺序**：物理护盾只吸收物理伤害，法术护盾只吸收法术伤害
3. **真实伤害**：无视护甲、魔抗、减伤效果，但可能被无敌状态免疫
4. **自适应伤害**：常用于英雄技能，可以自动适配对手的弱点
