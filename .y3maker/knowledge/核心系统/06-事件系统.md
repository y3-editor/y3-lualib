# Y3 事件系统

Y3 引擎的事件系统用于游戏逻辑的事件驱动编程。

## 📋 通用事件常量

### 怪物/战斗相关事件

| 事件名 | 说明 |
|--------|------|
| `E_MONSTER_DIE` | 怪物死亡事件 |
| `E_BATTLE_START` | 战斗开始 |
| `E_BATTLE_SETTLEMENT` | 战斗结算 |
| `E_BATTLE_READY` | 战斗准备完成 |
| `E_GAME_STATE_CHANGE` | 游戏状态变更 |

## 🔧 事件使用示例

### Lua 中监听事件

```lua
-- 监听怪物死亡事件
y3.event.on('E_MONSTER_DIE', function(event_data)
    local unit = event_data.unit
    print("怪物死亡: " .. tostring(unit))
end)

-- 监听技能使用
y3.event.on('E_USE_SKILL', function(event_data)
    local caster = event_data.caster
    local ability = event_data.ability
    print(caster:get_name() .. " 使用了技能")
end)
```

### 单位事件注册

```lua
-- 监听单位属性变更
local unit = y3.unit.get(unit_id)
unit:event('属性变化', function(event_data)
    local attr_name = event_data.attr_name
    local new_value = event_data.new_value
end)
```
