# 魔法效果物编 (Modifier/Buff)

**目录**：`maps/EntryMap/editor_table/modifierall/`

## 物编字段

```json
{
    // === 基础设置 ===
    "name": "效果名称",
    "description": "效果描述",
    "modifier_icon": 100008,       // 图标ID
    "show_on_ui": true,            // 是否在UI上显示
    
    // === 持续时间 ===
    "modifier_duration": -1,       // 持续时间（-1表示永久）
    "cycle_time": 0,               // 周期时间（秒）
    "disappear_when_dead": true,   // 死亡时消失
    
    // === 层数与覆盖 ===
    "layer_max": 1,                // 最大层数
    "layer_change_of_cover": 1,    // 覆盖时层数变化
    "modifier_cover_type": 1,      // 覆盖类型
    "same_origin_cover": true,     // 同源覆盖
    "time_change_of_cover": 1,     // 覆盖时时间变化
    
    // === 护盾 ===
    "shield_type": 1,              // 护盾类型
    "shield_value": 0.0,           // 护盾值
    "shield_change_of_cover": 2,   // 覆盖时护盾变化
    
    // === 控制效果 ===
    "forbid_move": false,          // 禁止移动（定身）
    "forbid_attack": false,        // 禁止普攻（缴械）
    "forbid_ability": false,       // 禁止技能（沉默）
    "is_debuff": false,            // 是否负面效果
    "is_purgable": false,          // 是否可被净化
    
    // === 影响范围（光环用） ===
    "influence_rng": 0,            // 影响范围(cm)
    "is_influence_self": true,     // 是否影响自身
    "target_allow": 1,             // 允许目标
    
    // === 特效 ===
    "get_effect_list": [],         // 获得时特效列表
    "lose_effect_list": [],        // 失去时特效列表
    "halo_effect": 0,              // 光环特效
    "attach_model_list": [],       // 附加模型列表
    
    // === 材质变化 ===
    "material_change": 1,
    "material_alpha": 0.5,
    "material_color": [255,255,255]
}
```

## 系统常量

### Buff影响类型 (ModifierEffectType)

| 类型 | 值 | 说明 |
|------|---|------|
| `NORMAL` | 0 | 正常（中性） |
| `POSITIVE` | 1 | 正面（增益） |
| `NEGATIVE` | 2 | 负面（减益） |

### Buff类型 (ModifierType)

| 类型 | 值 | 说明 |
|------|---|------|
| `NORMAL` | 1 | 普通Buff |
| `HALO` | 2 | 光环 |
| `HALO_EFFECT` | 3 | 光环效果 |
| `SHIELD` | 4 | 护盾 |

### 覆盖类型 (ModifierCoverType)

| 类型 | 值 | 说明 |
|------|---|------|
| `NOT_COVER` | 1 | 不覆盖 |
| `COVER_BY_HIGH_LEVEL` | 2 | 高等级覆盖 |
| `COVER_BY_NEW` | 3 | 新覆盖旧 |
| `COVER_BY_ADDTION` | 4 | 叠加层数 |

### 目标允许类型 (TargetAllowType)

| 类型 | 位掩码 | 说明 |
|------|--------|------|
| `SELF_CAMP` | 1 | 自身阵营 |
| `FRIEND_CAMP` | 2 | 友方阵营 |
| `ENEMY_CAMP` | 4 | 敌方阵营 |
| `NATURAL_CAMP` | 8 | 中立阵营 |

**常用组合**：自己=1, 自己+友军=3, 敌人=4, 所有=15

### 护盾类型 (ModifierShieldType)

| 类型 | 值 | 说明 |
|------|---|------|
| `COMMON` | 2 | 通用护盾（抵挡所有伤害） |
| `PHYSIC` | 0 | 物理护盾 |
| `MAGIC` | 1 | 法术护盾 |

## 属性修改

Buff可以通过 `attached_*` 字段修改单位属性：

| 字段前缀 | 说明 | 示例 |
|----------|------|------|
| `attached_` | 附加值修改 | `attached_attack_phy = 50` |
| `attached_ratio_` | 百分比修改 | `attached_ratio_hp_max = 0.2` |

### 常用属性修改

| 字段Key | 说明 |
|---------|------|
| `attached_hp_max` | 增加最大生命值 |
| `attached_attack_phy` | 增加物理攻击 |
| `attached_attack_mag` | 增加法术攻击 |
| `attached_defense_phy` | 增加物理防御 |
| `attached_defense_mag` | 增加法术防御 |
| `attached_ori_speed` | 增加移动速度 |
| `attached_ratio_ori_speed` | 百分比移动速度 |

  "is_debuff": false,

  "show_on_ui": true

## 触发器事件

| 事件 | 说明 |
|------|------|
| ET_MODIFIER_OBTAINS | Buff获得时 |
| ET_MODIFIER_LOSE | Buff失去时 |
| ET_MODIFIER_CYCLE | Buff周期触发 |
| ET_MODIFIER_CHANGE_LAYER | Buff层数变化时 |
