# 投射物 (Projectile)

## 概述

投射物是 Y3 编辑器中用于表现飞行弹道的对象，通常用于技能的视觉效果。投射物可以独立创建，也可以创建后关联单位或技能。投射物本身不具备运动和碰撞检测的能力，通常需要配合运动器实现。

**在游戏中的作用**：

- 实现技能的弹道效果（火球、箭矢、导弹等）
- 配合运动器实现追踪、直线、曲线等多种运动轨迹

**与其他物编的关系**：

- 投射物可以关联**技能 (Ability)**，用于伤害归属
- 投射物通过**运动器 (Mover)** 实现运动和碰撞

## 核心概念

### 物编 ID vs 运行时实例

```lua
-- 物编 ID：在编辑器中定义的投射物类型
local proj_key = 134267518  -- 例如：火球投射物的物编 ID

-- 运行时实例：游戏中创建的具体投射物对象
local proj = y3.projectile.create({
    key = proj_key,
    target = point,
    owner = unit
})
```

### 投射物与运动器

投射物本身只是一个视觉表现对象，其运动和碰撞逻辑由**运动器 (Mover)** 控制：

```lua
-- 创建投射物
local proj = y3.projectile.create({
    key = proj_key,
    target = point,
    owner = unit
})

-- 给投射物添加运动器
proj:mover_line({
    angle = 0,
    distance = 500,
    speed = 800,
    hit_radius = 100,
    hit_type = 0,
    terrain_block = true,
    on_hit = function(mover, target)
        -- 碰撞回调
    end
})
```

### 投射物的生命周期

1. **创建** - 通过 `y3.projectile.create()` 创建
2. **运动** - 通过运动器控制飞行轨迹
3. **碰撞** - 运动器检测碰撞并触发回调
4. **移除** - 到达目标或超时后移除

## 创建与获取

### 创建投射物

```lua
---@param data Projectile.CreateData
---@return Projectile
y3.projectile.create(data)
```

**参数说明**：
```lua
{
    key = proj_key,              -- 投射物物编 ID（必需）
    target = point,              -- 创建位置：Point 或 Unit（必需）
    socket = 'origin',           -- 单位挂接点名（target 为 Unit 时有效）
    angle = 0,                   -- 朝向（度）
    height = 0,                  -- 高度
    time = 60.0,                 -- 持续时间（秒）
    owner = unit,                -- 所属单位或玩家
    ability = ability,           -- 关联技能
    visible_rule = 1,            -- 可见性规则
    remove_immediately = false,  -- 是否立即移除表现
    show_in_fog = false          -- 是否在迷雾中显示
}
```

**示例**：
```lua
-- 在指定位置创建投射物
local point = y3.point.create(0, 0, 0)
local proj = y3.projectile.create({
    key = 134267518,
    target = point,
    owner = caster,
    angle = 90,
    time = 5.0
})

-- 在单位身上创建投射物
local proj = y3.projectile.create({
    key = 134267518,
    target = enemy,
    socket = 'chest',  -- 挂接到单位的 chest 挂接点
    owner = caster
})
```

### 获取投射物

```lua
-- 通过运行时 ID 获取
local proj = y3.projectile.get_by_id(proj_id)

-- 通过 py 层句柄获取
local proj = y3.projectile.get_by_handle(py_proj)

-- 通过物编对象获取（用于事件监听）
local proj_editor = y3.object.projectile[134267518]
```

## 运动器系统

投射物的运动和碰撞由运动器控制。运动器有四种类型：

> **on_hit 触发前置条件**：如果希望触发 `on_hit`，需要正确配置碰撞相关参数，至少包括：
> - `hit_type`（碰撞类型）
> - `hit_radius`（碰撞范围）
> - 是否与地形碰撞（如 `terrain_block`，按具体运动器支持情况设置）
>
> 若这些参数缺失或配置不当，`on_hit` 可能不会触发。

### 直线运动

```lua
proj:mover_line({
    angle = 0,                   -- 方向（度）
    distance = 500,              -- 距离
    speed = 800,                 -- 初始速度
    acceleration = 0,            -- 加速度
    max_speed = 1000,            -- 最大速度
    min_speed = 0,               -- 最小速度
    init_height = 0,             -- 初始高度
    fin_height = 0,              -- 终点高度
    parabola_height = 100,       -- 抛物线顶点高度
    hit_radius = 100,            -- 碰撞半径
    hit_type = 0,                -- 0:敌人 1:盟友 2:全部
    hit_same = false,            -- 能否重复碰撞同一单位
    terrain_block = true,        -- 是否地形阻挡
    face_angle = true,           -- 是否面向运动方向
    on_hit = function(mover, target)
        -- 碰撞单位时触发
        print('命中:', target:get_name())
    end,
    on_block = function(mover)
        -- 碰撞地形时触发
        print('碰到地形')
    end,
    on_finish = function(mover)
        -- 运动结束时触发
        print('运动结束')
    end
})
```

### 追踪运动

```lua
proj:mover_target({
    target = enemy,              -- 追踪目标（Unit/Destructible/Item）
    speed = 600,                 -- 初始速度
    target_distance = 50,        -- 撞击距离
    acceleration = 100,          -- 加速度
    max_speed = 1000,            -- 最大速度
    height = 0,                  -- 初始高度
    parabola_height = 50,        -- 抛物线高度
    bind_point = 'chest',        -- 绑定点
    missing_distance = 2000,     -- 目标丢失距离
    miss_when_target_destroy = true,  -- 目标销毁时丢失
    hit_radius = 80,
    hit_type = 0,
    on_hit = function(mover, target)
        -- 命中目标
    end,
    on_miss = function(mover)
        -- 目标丢失
        print('目标丢失')
    end,
    on_finish = function(mover)
        -- 追踪结束
    end
})
```

### 曲线运动

```lua
proj:mover_curve({
    path = { point1, point2, point3 },  -- 路径点列表
    speed = 500,                 -- 速度
```

## 事件监听

### 在物编对象上监听

适用于监听某个投射物类型的所有实例：

```lua
local proj_editor = y3.object.projectile[134267518]

-- 监听投射物创建
proj_editor:event('投射物-创建', function(trg, data)
    local proj = data.projectile
    print('创建了投射物:', proj:get_key())
end)

-- 监听投射物死亡
proj_editor:event('投射物-死亡', function(trg, data)
    local proj = data.projectile
    print('投射物死亡')
end)
```

### 在运行时实例上监听

适用于监听特定投射物实例的事件：

```lua
local proj = y3.projectile.create({
    key = proj_key,
    target = point,
    owner = unit
})

-- 监听投射物死亡
proj:event('投射物-死亡', function(trg, data)
    print('投射物已死亡')
end)
```

### 常用事件列表

| 事件名 | 触发时机 | 主要参数 |
|--------|----------|----------|
| `投射物-创建` | 投射物创建时 | `projectile` |
| `投射物-死亡` | 投射物死亡时 | `projectile` |

**注意**：投射物的碰撞事件通过运动器的回调函数实现，而非事件系统。

## 典型应用场景

### 场景 1：直线火球术

**需求描述**：
释放火球术，火球直线飞行 1000 距离，命中敌人造成伤害。

**实现代码**：
```lua
local ability_editor = y3.object.ability[134218450]  -- 火球术

ability_editor:event('施法-出手', function(trg, data)
    local caster = data.unit
    local target_point = data.target_point

    -- 计算方向
    local caster_pos = caster:get_point()
    local angle = caster_pos:get_angle_to(target_point)

    -- 创建投射物
    local proj = y3.projectile.create({
        key = 134267518,
        target = caster_pos,
        owner = caster,
        angle = angle,
        ability = data.ability
    })

    -- 添加直线运动器
    proj:mover_line({
        angle = angle,
        distance = 1000,
        speed = 800,
        hit_radius = 100,
        hit_type = 0,  -- 只碰撞敌人
        terrain_block = true,
        face_angle = true,
        on_hit = function(mover, target)
            -- 造成伤害
            caster:damage({
                target = target,
                damage = 200,
                type = '魔法'
            })

            -- 移除投射物
            proj:remove()
        end,
        on_block = function(mover)
            -- 碰到地形，移除投射物
            proj:remove()
        end,
        on_finish = function(mover)
            -- 飞行结束，移除投射物
            proj:remove()
        end
    })
end)
```

**说明**：
- 使用直线运动器实现火球飞行
- 碰撞敌人时造成伤害并移除
- 碰撞地形或飞行结束时移除

### 场景 2：追踪导弹

**需求描述**：
发射追踪导弹，自动追踪目标单位，命中后造成伤害和眩晕。

**实现代码**：
```lua
local ability_editor = y3.object.ability[134218451]  -- 追踪导弹

ability_editor:event('施法-出手', function(trg, data)
    local caster = data.unit
    local target = data.target_unit

    if not target then return end

    -- 创建投射物
    local proj = y3.projectile.create({
        key = 134267519,
        target = caster:get_point(),
        owner = caster,
        ability = data.ability
    })

    -- 添加追踪运动器
    proj:mover_target({
        target = target,
        speed = 600,
        acceleration = 200,
        max_speed = 1200,
        target_distance = 50,
        hit_radius = 80,
        hit_type = 0,
        missing_distance = 3000,
        miss_when_target_destroy = true,
        on_hit = function(mover, hit_target)
            -- 造成伤害
            caster:damage({
                target = hit_target,
                damage = 300,
                type = '魔法'
            })

            -- 添加眩晕 Buff
            hit_target:add_buff({
                key = 200001,  -- 眩晕 Buff
                source = caster,
                time = 2.0
            })

            -- 播放爆炸特效
            y3.effect.create_effect_on_unit('爆炸特效', hit_target, 1.0)

            -- 移除投射物
            proj:remove()
        end,
        on_miss = function(mover)
            -- 目标丢失，移除投射物
            print('目标丢失')
            proj:remove()
        end,
        on_finish = function(mover)
            -- 追踪结束
            proj:remove()
        end
    })
end)
```

**说明**：
- 使用追踪运动器实现自动追踪
- 设置加速度实现加速追踪效果
- 目标丢失或销毁时自动处理

### 场景 3：范围箭雨

**需求描述**：
向目标区域发射多支箭矢，每支箭矢呈抛物线飞行，命中敌人造成伤害。

**实现代码**：
```lua
local ability_editor = y3.object.ability[134218452]  -- 箭雨

ability_editor:event('施法-出手', function(trg, data)
    local caster = data.unit
    local target_point = data.target_point
    local caster_pos = caster:get_point()

    -- 发射 10 支箭矢
    for i = 1, 10 do
        -- 随机偏移目标点
        local offset_x = math.random(-200, 200)
        local offset_y = math.random(-200, 200)
        local final_point = y3.point.create(
            target_point.x + offset_x,
            target_point.y + offset_y,
            0
        )

        -- 计算方向和距离
        local angle = caster_pos:get_angle_to(final_point)
        local distance = caster_pos:get_distance_to(final_point)

        -- 延迟发射
        y3.ltimer.wait(i * 0.1, function()
            -- 创建箭矢投射物
            local proj = y3.projectile.create({
                key = 134267520,
                target = caster_pos,
                owner = caster,
                angle = angle
            })

            -- 添加抛物线运动
            proj:mover_line({
                angle = angle,
                distance = distance,
                speed = 1000,
                parabola_height = 300,  -- 抛物线高度
                hit_radius = 80,
                hit_type = 0,
                hit_same = false,  -- 不重复碰撞
                on_hit = function(mover, target)
                    -- 造成伤害
                    caster:damage({
                        target = target,
                        damage = 50,
                        type = '物理'
                    })
                end,
                on_finish = function(mover)
                    -- 箭矢落地，移除
                    proj:remove()
                end
            })
        end)
    end
end)
```

**说明**：
- 使用循环创建多个投射物
- 使用 `parabola_height` 实现抛物线效果
- 延迟发射实现箭雨效果

### 场景 4：环绕护盾

**需求描述**：
在单位周围创建环绕的能量球，持续旋转，碰到敌人造成伤害。

**实现代码**：
```lua
local ability_editor = y3.object.ability[134218453]  -- 环绕护盾

ability_editor:event('施法-完成', function(trg, data)
    local caster = data.unit

    -- 创建 3 个环绕能量球
    for i = 1, 3 do
        local proj = y3.projectile.create({
            key = 134267521,
            target = caster:get_point(),
            owner = caster,
            time = 10.0  -- 持续 10 秒
        })

        -- 添加环绕运动器
        proj:mover_round({
            target = caster,
            radius = 200,
            angle_speed = 120,  -- 120度/秒
            init_angle = i * 120,  -- 初始角度间隔 120 度
            clock_wise = true,
            round_time = 10.0,
            height = 100,
            hit_radius = 60,
            hit_type = 0,
            hit_same = true,  -- 可重复碰撞
            hit_interval = 1.0,  -- 碰撞间隔 1 秒
            on_hit = function(mover, target)
                -- 造成伤害
                caster:damage({
                    target = target,
                    damage = 30,
                    type = '魔法'
                })
            end,
            on_finish = function(mover)
                -- 环绕结束，移除
                proj:remove()
            end
        })

        -- 绑定到施法者，施法者死亡时自动清理
        caster:bindGC(proj)
    end
end)
```

**说明**：
- 使用环绕运动器实现旋转效果
- 设置初始角度实现均匀分布
- 使用 `hit_same` 和 `hit_interval` 实现重复碰撞

### 场景 5：曲线弹道

**需求描述**：
投射物沿着预定路径飞行，实现曲线弹道效果。

**实现代码**：
```lua
local ability_editor = y3.object.ability[134218454]  -- 曲线弹道

ability_editor:event('施法-出手', function(trg, data)
    local caster = data.unit
    local target_point = data.target_point
    local caster_pos = caster:get_point()

    -- 计算路径点（贝塞尔曲线的控制点）
    local mid_x = (caster_pos.x + target_point.x) / 2
    local mid_y = (caster_pos.y + target_point.y) / 2

    -- 控制点偏移，形成弧线
    local offset = 300
    local path = {
        caster_pos,
        y3.point.create(mid_x + offset, mid_y, 0),
        y3.point.create(mid_x, mid_y + offset, 0),
        target_point
    }

    -- 创建投射物
    local proj = y3.projectile.create({
        key = 134267522,
        target = caster_pos,
        owner = caster
    })

    -- 添加曲线运动器
    proj:mover_curve({
        path = path,
        speed = 600,
        hit_radius = 100,
        hit_type = 0,
        on_hit = function(mover, target)
            -- 造成伤害
            caster:damage({
                target = target,
                damage = 250,
                type = '魔法'
            })
            proj:remove()
        end,
        on_finish = function(mover)
            -- 到达终点
            proj:remove()
        end
    })
end)
```

**说明**：
- 使用曲线运动器实现复杂弹道
- 通过路径点控制飞行轨迹
- 可以实现绕过障碍物的效果

## 与其他物编的协作

### 与单位 (Unit) 的协作


### 与技能 (Ability) 的协作

```lua
-- 技能释放时创建投射物
ability:event('施法-出手', function(trg, data)
    local proj = y3.projectile.create({
        key = proj_key,
        target = data.target_point,
        owner = data.unit,
        ability = data.ability  -- 关联技能
    })
end)

-- 投射物获取关联技能
local ability = proj:get_ability()
```

### 与魔法效果 (Buff) 的协作

```lua
-- 投射物命中时添加 Buff
proj:mover_target({
    target = enemy,
    speed = 500,
    hit_radius = 80,
    hit_type = 0,
    on_hit = function(mover, target)
        target:add_buff({
            key = buff_key,
            source = caster,
            time = 5.0
        })
        proj:remove()
    end
})
```
