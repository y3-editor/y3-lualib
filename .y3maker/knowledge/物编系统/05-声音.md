# 声音 (Sound)

## 概述

声音对象用于播放、控制 2D/3D 音效。常见用途：

- UI 提示音
- 技能与战斗音效
- 场景环境音
- 跟随单位的动态音效

## 核心概念

### 声音物编 ID vs 运行时声音实例

```lua
-- 声音物编 ID（py.AudioKey）
local sound_key = 140196

-- 运行时实例（Sound）
local sound = y3.sound.play(player, sound_key)
```

`sound_key` 是编辑器中的配置类型；`Sound` 是一次播放产生的运行时对象，可继续调整音量或停止。

## 播放与控制

### 2D 播放

```lua
---@class Sound.PlayOptions
---@field loop? boolean
---@field fade_in? number
---@field fade_out? number

---@param player Player
---@param sound py.AudioKey
---@param options? Sound.PlayOptions
---@return Sound?
y3.sound.play(player, sound, options)
```

### 3D 播放（点）

```lua
---@class Sound.Play3DOptions: Sound.PlayOptions
---@field height? number
---@field ensure? boolean

---@param player Player
---@param sound py.AudioKey
---@param point Point
---@param options? Sound.Play3DOptions
---@return Sound?
y3.sound.play_3d(player, sound, point, options)
```

### 3D 播放（跟随单位）

```lua
---@class Sound.PlayUnitOptions: Sound.PlayOptions
---@field ensure? boolean
---@field offset_x? number
---@field offset_y? number
---@field offset_z? number

---@param player Player
---@param sound py.AudioKey
---@param unit Unit
---@param options? Sound.PlayUnitOptions
---@return Sound?
y3.sound.play_with_object(player, sound, unit, options)
```

### 运行时控制

```lua
-- 设置音量（0-100）
sound:set_volume(player, 80)

-- 停止播放
sound:stop(player, true)   -- 立即停止
sound:stop(player, false)  -- 非立即停止
```

## 示例

```lua
local player = y3.player.get_by_id(1)
local point = y3.point.create(0, 0, 0)
local unit = y3.unit.get_by_id(10001)

-- 2D 提示音
local ui_sound = y3.sound.play(player, 140196, {
    fade_in = 0.1,
    fade_out = 0.1,
    loop = false,
})

-- 3D 场景音
local scene_sound = y3.sound.play_3d(player, 140197, point, {
    height = 0,
    ensure = true,
    fade_in = 0.2,
    fade_out = 0.2,
})

-- 跟随单位
local follow_sound = y3.sound.play_with_object(player, 140198, unit, {
    ensure = true,
    loop = true,
    offset_z = 80,
})

-- 动态调节
if follow_sound then
    follow_sound:set_volume(player, 60)
end
```

## 注意事项

1. 声音是按玩家播放的，调用时必须传入 `Player`。
2. `play_*` 返回 `nil` 代表本次播放未成功，不应继续调用实例方法。
3. 大量循环音效要及时 `stop`，避免资源与听觉堆叠。
