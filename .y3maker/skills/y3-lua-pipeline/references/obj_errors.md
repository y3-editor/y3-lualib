# Y3 物编错误速查

记录物编 JSON 生成和配置常见错误。

---

## 🔴 致命错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 物编生成到 `.codemaker/` | 编辑器无法识别 | 生成到 `maps/EntryMap/editor_table/` |
| `uid` 是整数 `400001` | 类型错误 | 必须是字符串 `"400001"` |
| 技能列表用普通数组 | 解析失败 | `{"__tuple__": true, "items": [...]}` |
| `kv` 字段存自定义数据 | 编辑器崩溃 | 保持空对象 `{}` |
| 直接修改 `data_template/*.json` | 模板污染 | 只修改 `editor_table/` 输出文件 |

---

## 🟡 字段错误

### 技能释放类型
| 字段 | 说明 |
|------|------|
| `sight_type` | **释放类型**：`0`=无目标直接释放，`4`=需要选择单位目标 |
| `pointer_limitation` | 指示器类型（非释放类型）：`0`=无，`1`=扇形，`4`=目标，`5`=点 |

### 击杀奖励
| 错误字段 | 正确字段 |
|----------|----------|
| `dead_exp` | `reward_exp` |
| `dead_money` | `reward_official_res_1` |

### 物品属性加成
```json
// ✅ 正确（引擎自动管理，仅英雄生效）
{
  "attached_attack_phy": 50,
  "attached_hp_max": 100
}
```
