# 技能物编 (Ability)

**目录**：`maps/EntryMap/editor_table/abilityall/`

## 技能释放流程

```
施法-即将开始 → 施法-开始 → 施法-引导 → 施法-出手 → 施法-完成 → 施法-结束
                  ↓            ↓
               前摇开始      前摇结束
                           扣除资源
                           进入冷却
```

| 事件 | 说明 |
|------|------|
| `施法-开始` (ET_ABILITY_PS_START) | 施法开始，**前摇开始** |
| `施法-引导` (ET_ABILITY_PS_END) | **前摇结束**，资源扣除、冷却开始 |
| `施法-出手` (ET_ABILITY_SP_END) | **技能效果生效**，后摇开始 |
| `施法-完成` (ET_ABILITY_BS_END) | 后摇结束 |

> **推荐触发时机**：`施法-出手` 是技能效果生效的最佳时机

## 物编字段

```json
{
    // === 基础设置 ===
    "name": "技能名称",
    "ability_icon": 100508,        // 技能图标ID
    "ability_max_level": 1,        // 最大等级
    "ability_cast_type": 2,        // 施法类型
    
    // === 施法参数（数组按等级） ===
    "ability_cast_range": ["10.0"],     // 施法距离(cm)
    "ability_cast_point": 0.2,          // 施法前摇(秒)
    "ability_bw_point": 0.5,            // 施法后摇(秒)
    "ability_channel_time": 0.0,        // 持续施法时间
    "ability_prepare_time": 0.0,        // 准备时间
    
    // === 消耗与冷却 ===
    "cold_down_time": ["5"],            // 冷却时间（秒）
    "ability_cost": ["5"],              // 魔法消耗
    "ability_hp_cost": ["0"],           // 生命消耗
    "influenced_by_cd_reduce": true,    // 是否被冷却缩减影响
    
    // === 伤害 ===
    "ability_damage": ["5"],            // 伤害值
    "ability_damage_range": ["0.0"],    // 伤害范围(cm)
    
    // === 指示器形状 ===
    "sight_type": 0,                    // 指示器类型
    "circle_radius": ["5.0"],           // 圆形半径
    "sector_radius": ["10.0"],          // 扇形半径
    "sector_angle": ["0"],              // 扇形角度
    "arrow_length": ["5.0"],            // 箭头长度
    "arrow_width": ["2.0"],             // 箭头宽度
    
    // === 目标过滤 ===
    "filter_condition_camp": 5,         // 阵营过滤（位掩码）
    "filter_condition_type": 7,         // 类型过滤（位掩码）
    "filter_unit_tags": [],             // 过滤单位标签
    
    // === 打断设置 ===
    "can_cast_interrupt": false,        // 施法可打断
    "can_prepare_interrupt": false,     // 准备可打断
    "can_bs_interrupt": true,           // 后摇可打断
    "can_ps_interrupt": true,           // 前摇可打断
    
    // === 充能技能 ===
    "is_charge_ability": false,         // 是否充能技能
    "ability_max_stack_count": ["0"],   // 最大充能层数
    "ability_stack_cd": ["0"],          // 充能冷却
    
    // === 施法行为 ===
    "is_immediate": false,              // 是否即时
    "is_autocast": false,               // 是否自动施法
    "need_turn_to_target": true         // 需要转向目标
}
```

## 施法类型枚举 (ability_cast_type)

| 值 | 类型 | 说明 |
|----|------|------|
| 1 | 无目标 | 立即释放，无需选择目标 |
| 2 | 单位目标 | 需要选择一个单位目标 |
| 3 | 点目标 | 需要选择一个位置 |
| 4 | 点+单位目标 | 可选择点或单位 |
| 5 | 方向目标 | 选择一个方向释放 |
| 6 | 向量目标 | 选择起点和终点 |
| 7 | 建造目标 | 选择建造位置 |
| 8 | 物品目标 | 选择一个物品 |
| 9 | 可破坏物目标 | 选择一个可破坏物 |

## 指示器类型 (sight_type)

| 值 | 类型 | 适用场景 |
|----|------|----------|
| 0 | 无指示器 | 无目标技能 |
| 1 | 圆形 | AOE技能 |
| 2 | 扇形 | 锥形技能 |
| 3 | 箭头 | 方向技能 |

## 技能释放错误码

技能释放失败时返回的错误码：

### 基础错误

| 错误码 | 常量名 | 说明 |
|--------|--------|------|
| 1 | NO_ERR | 无错误 |
| 2 | ABILITY_NOT_EXIST | 技能不存在 |
| 3 | ABILITY_COOLING | 技能冷却中 |
| 4 | MP_NOT_ENOUGH | 魔法值不足 |
| 5 | HP_NOT_ENOUGH | 生命值不足 |
| 6 | UNIT_IS_DEAD | 单位已死亡 |
| 7 | UNIT_IS_SILENCE | 单位被沉默 |

### 目标错误

| 错误码 | 常量名 | 说明 |
|--------|--------|------|
| 10 | NO_TARGET | 没有目标 |
| 11 | TARGET_OUT_OF_RANGE | 目标超出范围 |
| 12 | TARGET_IS_DEAD | 目标已死亡 |
| 13 | TARGET_IS_IMMUNE | 目标免疫 |
| 14 | TARGET_CAMP_INVALID | 目标阵营不符 |
| 15 | TARGET_TYPE_INVALID | 目标类型不符 |

### 状态错误

| 错误码 | 常量名 | 说明 |
|--------|--------|------|
| 20 | UNIT_IS_MOVING | 单位移动中（移动打断） |
| 21 | UNIT_IS_CASTING | 单位施法中 |
| 22 | ABILITY_DISABLED | 技能被禁用 |
| 23 | UNIT_IS_STUNED | 单位被眩晕 |

## 目标筛选

### 阵营过滤 (filter_condition_camp)

| 位掩码 | 阵营 |
|--------|------|
| 1 | 己方 |
| 2 | 友方 |
| 4 | 敌方 |
| 8 | 中立 |

**常用组合**：敌人=4, 己方+友方=3, 所有=15

### 类型过滤 (filter_condition_type)

| 位掩码 | 类型 |
|--------|------|
| 1 | 英雄 |
| 2 | 建筑 |
| 4 | 生物 |
| 8 | 可破坏物 |
| 16 | 物品 |

**常用组合**：所有单位=7, 建筑除外=5
