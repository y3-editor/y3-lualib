## ⚠️ 重要提示

1. **UI 坐标系**：Y3 原点在**左下角**（Y↑ X→），与常规 GUI 左上角不同
2. **击杀奖励**：使用 `reward_exp` 和 `reward_official_res_1`，而非 `dead_exp`
3. **官方组件**：技能栏/Buff/物品槽使用 `type:17/18/20`，不要手写
4. **物编保存**：生成物编后必须**先热更再保存**，否则内容丢失
