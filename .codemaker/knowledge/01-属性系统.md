# 属性系统

## 1. 单位属性 (Unit Attributes)

**文件位置**：
- `maps\EntryMap\attr.json` - **主要文件**（地图级）
- `attr.json` - 根目录（暂时无意义，保持一致防出错，未来可能支持项目级属性）

**数据结构**：

```json
{
    "p": [],    // Y3 预设属性（RTS基础属性）
    "c": [],    // 自定义属性（用户添加）
    "b": [],    // 待确认
    "ps": 1     // 待确认
}
```

**预设属性 `"p"`**（Y3 内置 RTS 属性）：
- `strength` - 力量
- `agility` - 敏捷
- `intelligence` - 智力
- `main` - 主属性

**自定义属性 `"c"` 字段结构**：

```json
{
    "__tuple__": true,
    "items": [
        "okmAOCHQEfGPIjkspskq7h",  // key（系统生成的唯一标识符）
        {
            "key": "okmAOCHQEfGPIjkspskq7h",
            "desc": "test",           // 属性名称/描述
            "type": "Fixed",          // 数据类型
            "default": 0.0,           // 默认值
            "minimun": 0.0,           // 最小值
            "maximun": 70000000000000, // 最大值
            "desc_bit": 1             // 描述位标记
        }
    ]
}
```

## 2. 玩家属性 (Player/Role Attributes)

**文件位置**：`campinfo.json`

**数据结构**：

```json
{
    "role_res_types": [],  // 玩家属性/资源类型定义
    "roles": [],           // 玩家配置
    "camps": []            // 阵营配置
}
```

**玩家属性 `role_res_types` 字段结构**：

```json
{
    "__tuple__": true,
    "items": [
        "poylqiHQEfGcKzkspskq7p",  // key
        {
            "key": "poylqiHQEfGcKzkspskq7p",
            "name": "role2",           // 属性名称
            "is_coin": true,           // 是否为货币类型
            "icon_id": 134227314,      // 货币图标ID（非货币为null）
            "default": 0.0,            // 默认值
            "lower_bound": 0.0,        // 下限
            "upper_bound": 70000000000000  // 上限
        }
    ]
}
```

**货币 vs 普通属性**：

| 类型 | is_coin | icon_id |
|------|---------|---------|
| 货币 | true | 有值（图标ID） |
| 普通 | false | null |
