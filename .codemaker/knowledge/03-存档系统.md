# 玩家存档系统 (Archive System)

## 概述

玩家存档系统用于持久化存储玩家数据，支持多种数据类型和排行榜功能。

**来源**：`E:\trunk分支\src\Server\server\engine\dm\commons\black_box\consts\role_consts.py` → `ArchiveType`

**文件位置**：
- `archive\archive.json` - 项目级（与地图级保持一致）
- `maps\EntryMap\archive.json` - **地图级**（主要文件）

## 数据结构

```json
{
    "archive_slots": {},           // 存档槽位定义
    "archive_drop_config": {},     // 存档掉落配置
    "multi_keys_limit_config": {}, // 多键限制配置
    "open_archive_key": 0          // 开放存档键
}
```

## 存档槽位字段

```json
{
    "1": {
        "key": 1,                        // 槽位ID（整数键）
        "name": "玩家名",                 // 存档名称
        "desc": "",                      // 描述
        "type": 0,                       // 存档类型（见下表）
        "value": "",                     // 默认值
        "encrypt": true,                 // 是否加密（防作弊）
        "game_mode_limit": 0,            // 游戏模式限制
        "max_value_limit_type": 0,       // 最大值限制类型
        "billboard_max_item_num": 0,     // 排行榜最大项数
        "billboard_sort_type": 0,        // 排行榜排序类型
        "time_interval_limit": [],       // 时间间隔限制
        "week_date_limit": []            // 每周日期限制
    }
}
```

## 存档类型 (ArchiveType)

| type 值 | 枚举名称 | 中文名称 | 默认值 |
|--------|---------|---------|--------|
| 0 | STR | 字符串 | "" |
| 1 | BOOL | 布尔 | 0 |
| 2 | INT | 整数 | 0 |
| 3 | FLOAT | 实数 | 0.0 |
| 4 | TABLE | 表格 | "{}" |

**数值类型**：`NUMERIC_TYPES = (INT, FLOAT)` - 可用于数值运算

## 全局存档类型 (GlobalArchiveType)

| type 值 | 枚举名称 |
|--------|---------|
| 0 | INT |
| 1 | STR |

## 存档存储类型 (LocalArchiveStorageType)

| 类型 | 说明 |
|------|------|
| PLAYER_ARCHIVE | 玩家存档 |
| PLATFORM_PROP | 平台道具 |
| RANK | 排行榜 |
| GLOBAL_ARCHIVE | 全局存档 |
