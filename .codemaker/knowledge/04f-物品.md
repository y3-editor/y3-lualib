# 物品物编 (Item)

**目录**：`editoritem/`

## 专属字段

```json
{
    // === 基础设置 ===
    "icon": 100000,                 // 图标ID
    "model": 10022,                 // 掉落物模型ID
    "body_size": 1.0,               // 体型大小
    "level": 1,                     // 物品等级
    
    // === 生命值 ===
    "hp_max": 100,                  // 最大生命值（掉落物）
    
    // === 堆叠与充能 ===
    "cur_stack": 1,                 // 当前堆叠数
    "maximum_stacking": 5,          // 最大堆叠数
    "stack_type": 2,                // 堆叠类型
    "cur_charge": 0,                // 当前充能数
    "maximum_charging": 5,          // 最大充能数
    
    // === 使用设置 ===
    "use_consume": 1,               // 使用消耗（0=不消耗，1=消耗1个）
    "auto_use": false,              // 自动使用
    
    // === 丢弃设置 ===
    "discard_enable": true,         // 可丢弃
    "discard_when_dead": true,      // 死亡时丢弃
    "delete_on_discard": false,     // 丢弃时删除
    "drop_stay_time": 9999,         // 掉落物存留时间（秒）
    
    // === 出售设置 ===
    "sale_enable": true,            // 可出售
    "buy_res_list": [],             // 购买所需资源
    "sell_res_list": [],            // 出售获得资源
    
    // === 库存设置 ===
    "init_stock": 5,                // 初始库存
    "max_stock": 5,                 // 最大库存
    "refresh_interval": 1,          // 刷新间隔
    "start_rft": 1,                 // 开始刷新时间
    
    // === 附加属性 ===
    "attached_ability": [],         // 附加技能
    "attached_passive_abilities": [], // 附加被动技能
    "attached_hp_max": 0.0,         // 附加最大生命
    "attached_hp_rec": 0.0,         // 附加生命恢复
    "attached_strength": 0.0,       // 附加力量
    "attached_agility": 0.0,        // 附加敏捷
    "attached_intelligence": 0.0,   // 附加智力
    "attached_critical_chance": 0.0, // 附加暴击率
    "attached_dodge_rate": 0.0,     // 附加闪避率
    "attached_hit_rate": 0.0,       // 附加命中率
    "attached_pene_phy": 0.0,       // 附加物理穿透
    "attached_pene_mag": 0.0,       // 附加魔法穿透
    
    // === 合成 ===
    "compose_list": [],             // 合成配方
    "precondition_list": [],        // 前置条件
    
    // === 其他设置 ===
    "disable_overlapping": false,   // 禁止叠加
    "cd_type": "",                  // 冷却类型
    "item_billboard_type": 0,       // 头顶板类型
    
    // === 点击交互 ===
    "click_name_pick_up": 0,        // 点击名称拾取
    "preview_name_click_range": 0,  // 预览名称点击范围
    "preview_name_stack": 0,        // 预览名称堆叠
    
    // === 变换 ===
    "offset": [0, 0, 0],            // 偏移
    "rotation": [0, 0, 0],          // 旋转
    
    // === 渲染设置 ===
    "model_opacity": 1,
    "material_change": 0,
    "material_alpha": 0.5,
    "material_color": [255, 255, 255],
    "material_color_intensity": 0,
    "base_color_mod": 1,
    "base_tint_color": [255, 255, 255, 0.5],
    "use_base_tint_color": false,
    "use_fresnel": 0,
    "fresnel_color": [255, 255, 255, 0.5],
    "fresnel_exp": 10,
    "fresnel_emissive_color_strength": 1,
    "use_virtual_light": 0
}
```

## 堆叠类型 (stack_type)

| 值 | 说明 |
|----|------|
| 0 | 不可堆叠 |
| 1 | 同ID堆叠 |
| 2 | 同类型堆叠 |

## 资源列表格式 (buy_res_list / sell_res_list)

```json
{
    "__tuple__": true,
    "items": [
        {
            "res_key": "official_res_1",  // 资源key
            "res_value": 100              // 资源数量
        }
    ]
}
```

## 合成配方格式 (compose_list)

```json
{
    "__tuple__": true,
    "items": [
        {
            "item_key": 100001,  // 材料物品ID
            "item_num": 2        // 材料数量
        }
    ]
}
```
