# 物编系统 (Editor Object System)

## 物编目录结构

**根路径**：`maps\EntryMap\editor_table\`

| 物编类型 | 目录名称 | 说明 |
|---------|---------|------|
| 单位 | `editorunit` | 单位物编 |
| 技能 | `abilityall` | 技能物编 |
| 物品 | `editoritem` | 物品物编 |
| 魔法效果(Buff) | `modifierall` | Buff/增益/减益效果 |
| 投射物 | `projectileall` | 投射物物编 |
| 可破坏物 | `editordestructible` | 可破坏物物编 |
| 声音 | `soundall` | 声音物编 |

**命名规则不统一**：
- `editor*` 前缀：`editorunit`, `editoritem`, `editordestructible`
- `*all` 后缀：`abilityall`, `modifierall`, `projectileall`, `soundall`

## 物编通用字段

所有物编 JSON 都包含以下通用字段：

```json
{
    "_custom_": true,              // 是否为自定义物编
    "_ref_": 134268131,            // 引用ID
    "key": 134268131,              // 物编ID（整数）
    "uid": "134268131",            // 物编ID（字符串）
    "name": -1186863046,           // 名称（语言包ID）
    "description": 230118278,      // 描述（语言包ID）
    "suffix": "",                  // 后缀
    "kv": {},                      // 自定义键值
    "tags": {                      // 标签列表
        "__tuple__": true, 
        "items": []
    },
    "sound_event_list": {          // 声音事件列表
        "__tuple__": true, 
        "items": []
    }
}
```

## 附加模型配置结构

用于魔法效果等物编的 `attach_model_list`：

```json
{
    "model": 103014,               // 模型ID
    "socket": "origin",            // 挂点名称
    "has_anim": true,              // 是否有动画
    "mount_attach": false,         // 是否挂载附加
    "offset": [0.0, 0.0, 0.0],     // 偏移
    "rotate": [0.0, 0.0, 0.0],     // 旋转
    "scale": [1.0, 1.0, 1.0],      // 缩放
    "offset_follow_scale": true,   // 偏移跟随缩放
    "size_follow_scale": true,     // 大小跟随缩放
    "follow_rotate": 0,            // 跟随旋转
    "anim_relate": {
        "default_anim": "None",
        "is_loop": true,
        "anim_map_list": []
    },
    "model_opacity": 1.0,
    "base_color_mod": 0,
    "base_tint_color": [255, 255, 255, 0.5],
    "use_base_tint_color": false,
    "fresnel_color": [255, 255, 255, 0.5],
    "fresnel_exp": 10.0,
    "fresnel_emissive_color_strength": 1.0,
    "use_fresnel": false,
    "use_virtual_light": false
}
```

## 特效配置结构

用于各种特效列表（如 `get_effect_list`、`effect_list`）：

```json
[
    100000,                        // 特效ID
    [0.0, 0.0, 0.0],              // 偏移 (x, y, z)
    [0.0, 0.0, 0.0],              // 旋转 (x, y, z)
    [1.0, 1.0, 1.0],              // 缩放 (x, y, z)
    1.0,                           // 透明度
    "origin",                      // 挂点名称
    true,                          // 参数1
    true,                          // 参数2
    0,                             // 参数3
    false                          // 参数4
]
```
