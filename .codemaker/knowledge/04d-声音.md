# 声音物编 (Sound)

**目录**：`soundall/`

## 专属字段

```json
{
    // === 基础设置 ===
    "icon": 100000,                // 图标ID
    "type": 1,                     // 声音类型
    "group": 0,                    // 声音组
    
    // === 声音配置 ===
    "sound_id": 1,                 // 声音ID（单个）
    "sound_id_list": [             // 声音ID列表（多个随机）
        {
            "sound_id": 140196,    // 声音资源ID
            "volume": 100,         // 音量
            "probability": 100     // 播放概率
        }
    ],
    
    // === 播放设置 ===
    "volume": 100,                 // 总音量
    "speed": 1,                    // 播放速度
    "priority": 1.0,               // 优先级
    "total_time": 0.417361,        // 总时长（秒）
    
    // === 淡入淡出 ===
    "fade_in_time": 0,             // 淡入时间
    "fade_out_time": 0,            // 淡出时间
    
    // === 距离衰减 ===
    "min_dist": 0,                 // 最小距离（开始衰减）
    "max_dist": 1000               // 最大距离（完全衰减）
}
```
