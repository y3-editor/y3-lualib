# 单位物编 (Unit)

**目录**：`maps/EntryMap/editor_table/editorunit/`

## 单位类型 (LogicUnitType)

位掩码系统，可组合使用：

| 类型 | 位掩码 | 说明 |
|------|--------|------|
| HERO | 1 | 英雄 |
| BUILDING | 2 | 建筑 |
| CREATURE | 4 | 生物/小怪 |
| DESTRUCTIBLE | 8 | 可破坏物 |
| ITEM | 16 | 地面物品 |
| SUMMON | 32 | 召唤物 |
| NPC | 64 | NPC |

**常用组合**：所有单位=7(1+2+4), 可攻击目标=15

---

## 物品栏系统

### 英雄单位有两种物品存放位置

| 类型 | SlotType | 属性生效 | 说明 |
|------|----------|----------|------|
| 物品栏 (BAR) | 1 | ✅ 生效 | 装备位 |
| 背包栏 (PKG) | 0 | ❌ 不生效 | 仓库位 |

> ⚠️ 只有**物品栏（BAR）**中的物品属性才会生效！

### 相关字段
| 字段 | 说明 |
|------|------|
| `bar_cnt` | 物品栏格数（默认 6） |
| `pkg_cnt` | 背包栏格数（默认 6） |

---

## 常用属性字段

### 战斗属性

| 字段 | 类型 | 说明 | 成长字段 |
|------|------|------|----------|
| `hp_max` | Float | 最大生命值 | `hp_max_grow` |
| `mp_max` | Float | 最大魔法值 | `mp_max_grow` |
| `hp_rec` | Float | 生命恢复/秒 | `hp_rec_grow` |
| `mp_rec` | Float | 魔法恢复/秒 | `mp_rec_grow` |
| `attack_phy` | Float | 物理攻击力 | `attack_phy_grow` |
| `attack_mag` | Float | 法术攻击力 | `attack_mag_grow` |
| `defense_phy` | Float | 物理防御 | `defense_phy_grow` |
| `defense_mag` | Float | 法术防御 | `defense_mag_grow` |
| `attack_speed` | Float | 攻击速度 | `attack_speed_grow` |
| `attack_range` | Float | 攻击范围(cm) | `attack_range_grow` |
| `attack_interval` | Float | 攻击间隔(秒) | `attack_interval_grow` |
| `ori_speed` | Float | 移动速度(cm/s) | `ori_speed_grow` |
| `critical_chance` | Float | 暴击率 | - |
| `critical_dmg` | Float | 暴击伤害(倍数) | - |
| `cd_reduce` | Float | 冷却缩减(%) | - |

### 穿透属性

| 字段 | 说明 |
|------|------|
| `pene_phy` | 物理穿透（固定值） |
| `pene_phy_ratio` | 物理穿透（百分比） |
| `pene_mag` | 法术穿透（固定值） |
| `pene_mag_ratio` | 法术穿透（百分比） |

### 吸血属性

| 字段 | 说明 |
|------|------|
| `vampire_phy` | 物理吸血(%) |
| `vampire_mag` | 法术吸血(%) |

### 视野属性

| 字段 | 说明 |
|------|------|
| `vision_rng` | 白天视野(cm) |
| `vision_night` | 夜晚视野(cm) |
| `vision_true` | 真实视野(反隐) |

### 英雄三维

| 字段 | 说明 | 成长字段 |
|------|------|----------|
| `strength` | 力量 | `strength_grow` |
| `agility` | 敏捷 | `agility_grow` |
| `intelligence` | 智力 | `intelligence_grow` |
| `main_attr` | 主属性 | 1=力量, 2=敏捷, 3=智力 |

### 击杀奖励

| 字段 | 说明 |
|------|------|
| `reward_exp` | 击杀经验奖励 |
| `reward_official_res_1` | 金币奖励（资源类型1） |

### 行为配置

| 字段 | 说明 | 选项 |
|------|------|------|
| `default_behaviour_type` | 默认行为 | 1=待机, 2=主动, 3=警戒 |
| `common_atk_type` | 攻击类型 | 1=近战, 2=远程, 3=不攻击 |
| `move_channel` | 移动通道 | 1=地面, 2=飞行 |

### 技能配置

| 字段 | 说明 |
|------|------|
| `hero_ability_list` | 英雄技能列表 |
| `common_ability_list` | 普通技能列表 |
| `passive_ability_list` | 被动技能列表 |

## Lua API

```lua
-- 获取物品栏中的物品
local items = hero:get_all_items_in_slot(y3.const.SlotType.BAR)

-- 获取背包栏中的物品
local pkg_items = hero:get_all_items_in_slot(y3.const.SlotType.PKG)

-- UI 绑定物品槽
slot:set_ui_unit_slot(hero, y3.const.SlotType.BAR, index)
```
