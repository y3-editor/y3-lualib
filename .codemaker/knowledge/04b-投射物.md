# 投射物物编 (Projectile)

**目录**：`projectileall/`

## 专属字段

```json
{
    // === 基础设置 ===
    "icon": 100000,                // 图标ID
    "max_duration": 60,            // 最大持续时间
    "poolable": 0,                 // 是否可池化
    
    // === 移动设置 ===
    "move_channel": 0,             // 移动通道
    "move_limitation": 1,          // 移动限制
    
    // === 特效设置 ===
    "effect_friend": [],           // 友方特效配置
    "effect_foes": [],             // 敌方特效配置
    "async_effect": false,         // 异步特效
    "effect_destroy_way_is_immediately": false, // 特效是否立即销毁
    "sfx_loop": false              // 特效是否循环
}
```

## 特效配置结构

`effect_friend` / `effect_foes` 结构：

```json
[
    106039,                        // 特效ID
    [0.0, 0.0, 0.0],              // 偏移
    [0.0, 0.0, 0.0],              // 旋转
    [1.0, 1.0, 1.0],              // 缩放
    1.0,                           // 透明度
    {                              // 模型配置
        "anim": "idle1",           // 动画名
        "model": 103977,           // 模型ID
        "is_loop": true,           // 是否循环
        "model_opacity": 1.0,      // 模型透明度
        "offset": {},              // 偏移
        "rotate": {},              // 旋转
        "scale": {},               // 缩放
        "base_color_mod": 1,
        "base_tint_color": [255, 255, 255, 0.5],
        "use_base_tint_color": false
    }
]
```
