# 镜头动画系统 (Camera Animation System)

## 概述

镜头动画系统用于创建摄像机过场动画、镜头移动效果等。

**文件位置**：`maps\EntryMap\cameraanim.json`

## 数据结构

```json
{
    "动画ID": {
        "anim_id": 1,                    // 动画ID
        "anim_name": null,               // 动画名称（语言包ID）
        "enabled": true,                 // 是否启用
        "return_first_frame": false,     // 结束后返回第一帧
        "use_game_last_frame": false,    // 使用游戏最后一帧
        "track_data": {}                 // 轨道数据
    }
}
```

## 轨道系统 (Track)

镜头动画采用**轨道-关键帧**结构：

```json
{
    "track_id": 1,               // 轨道ID
    "track_name": "Camera",      // 轨道名称
    "track_desc": "code_camline", // 轨道描述（语言包key）
    "track_type": 0,             // 轨道类型
    "is_loop": false,            // 是否循环
    "frame_data": {},            // 关键帧数据
    "sub_tracks": {}             // 子轨道
}
```

### 轨道类型 (track_type)

| track_type | track_name | track_desc | 说明 |
|------------|------------|------------|------|
| 0 | Camera | code_camline | 主摄像机轨道（容器） |
| 1 | Focus | code_attr_camera_focus | 焦点位置（Vector3） |
| 2 | Distance | code_attr_camera_dist | 摄像机距离 |
| 3 | Pitch | code_attr_camera_pitch | 俯仰角（弧度） |
| 4 | Roll | code_attr_camera_roll | 翻滚角（弧度） |
| 5 | Yaw | code_attr_camera_yaw | 偏航角（弧度） |
| 6 | Fov | code_attr_camera_fov | 视野角度 |
| 7 | FarClipping | code_attr_camera_far | 远裁剪面 |

### 轨道层级结构

```
Camera (track_type=0, 主轨道)
├── Focus (track_type=1, 焦点)
├── Distance (track_type=2, 距离)
├── Pitch (track_type=3, 俯仰)
├── Roll (track_type=4, 翻滚)
├── Yaw (track_type=5, 偏航)
├── Fov (track_type=6, 视野)
└── FarClipping (track_type=7, 远裁剪)
```

## 关键帧数据 (frame_data)

```json
{
    "帧号": {
        "frame_type": -1,        // 帧类型（-1=起始帧，0=普通帧，1=结束帧）
        "frame_value": 30.0,     // 帧值
        "value_type": 1,         // 值类型（1=数值，2=Vector3）
        "curve_type": 0,         // 曲线类型（插值方式）
        "curve_id": null         // 曲线ID（自定义曲线）
    }
}
```

### 帧类型 (frame_type)

| frame_type | 说明 |
|------------|------|
| -1 | 起始帧 |
| 0 | 普通帧 |
| 1 | 结束帧 |

### 值类型 (value_type)

| value_type | 说明 | 示例 |
|------------|------|------|
| 1 | 数值 | `30.0` |
| 2 | Vector3 | `{"__tuple__": true, "items": [x, y, z]}` |

### 曲线类型 (curve_type)

| curve_type | 说明 |
|------------|------|
| 0 | 线性插值 |

## 示例：镜头拉远动画

你创建的动画（ID=1）：
- **帧数**：0 到 26（约 0.87 秒，按 30fps）
- **效果**：Distance 从 30 拉远到 164.2
- **其他属性保持不变**：Focus、Pitch、Roll、Yaw、Fov、FarClipping

```
帧 0:  Distance = 30.0 (近景)
帧 26: Distance = 164.2 (远景)
```

## 运行时 API

```lua
-- 播放镜头动画
y3.camera.play_animation(player, anim_id)

-- 停止镜头动画
y3.camera.stop_animation(player)
```
