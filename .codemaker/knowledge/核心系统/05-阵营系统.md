# 阵营与玩家系统 (Camp & Role System)

## 概述

Y3 引擎的阵营系统用于管理玩家之间的敌对/友好关系。

**文件位置**：`campinfo.json`

**来源**：`E:\trunk分支\src\Server\server\engine\dm\commons\black_box\consts\role_consts.py`

## 数据结构

```json
{
    "camps": [],           // 阵营列表
    "roles": [],           // 玩家列表
    "role_res_types": []   // 玩家资源类型（见 01-属性系统.md）
}
```

## 阵营配置 (camps)

```json
{
    "__tuple__": true,
    "items": [
        1,                        // 阵营ID
        {
            "id": 1,              // 阵营ID
            "name": "$faction 1", // 阵营名称（$前缀表示语言包引用）
            "role_ids": [1],      // 所属玩家ID列表
            "share_vision": true  // 共享视野
        }
    ]
}
```

### 特殊阵营

| 阵营ID | 名称 | 说明 |
|--------|------|------|
| 1 | 阵营1 | 默认阵营 |
| 30 | 裁判 | 系统阵营 (`$code_camp_referee`) |
| 9999 | 中立阵营 | 存放中立玩家（31、32） |

## 玩家配置 (roles)

```json
{
    "__tuple__": true,
    "items": [
        1,                        // 玩家ID
        {
            "id": 1,              // 玩家ID
            "name": "$player 1",  // 玩家名称
            "camp_id": 1,         // 所属阵营ID
            "color": "#FF0000",   // 玩家颜色（十六进制）
            "type": 1             // 玩家类型
        }
    ]
}
```

### 玩家类型 (type)

| type值 | 说明 |
|--------|------|
| 1 | 普通玩家 |
| 2 | 中立玩家 |

### 特殊玩家

| 玩家ID | 类型 | 阵营 | 说明 |
|--------|------|------|------|
| 1-12 | 普通(1) | 可配置 | 真实玩家 |
| 31 | 中立(2) | 9999 | **中立敌对** - 攻击所有玩家（用于怪物） |
| 32 | 中立(2) | 9999 | **中立友好** - 不攻击任何人（用于NPC） |

## 关系类型

### RelationType（玩家间关系）

```python
class RelationType(object):
    INVALID = -1    # 无效
    ENEMY = 0       # 敌对
    FRIEND = 1      # 友好
```

**规则**：
- 初始化时：同阵营 = 友好，不同阵营 = 敌对
- 可通过 API 动态修改关系

### CampRelationType（阵营间关系）

```python
class CampRelationType(object):
    ENEMY = 1       # 敌方
    FRIEND = 2      # 友军
    NEUTRAL = 4     # 中立
    SELF = 8        # 自身
```

## 默认敌对规则

**核心规则**：
- 同阵营 → 友好
- 不同阵营 → 敌对

| 玩家A | 玩家B | 默认关系 | 说明 |
|-------|-------|----------|------|
| 1-12 | 同阵营 | **友好** | 同阵营默认友好 |
| 1-12 | 不同阵营 | **敌对** | 不同阵营默认敌对 |
| 31 | 1-12 | **敌对** | 中立敌对（阵营9999）与其他阵营敌对 |
| 31 | 32 | **友好** | 同属阵营9999，默认友好 |
| 32 | 1-12 | **友好** | 中立友好不主动攻击 |
| 32 | 31 | **友好** | 同属阵营9999，默认友好 |

**注意**：关系是**双向独立**的
- 玩家 31 对 1-12 敌对 → 31 会主动攻击 1-12
- 玩家 32 对 31 友好 → 32 不会主动攻击 31
- 但如果 31 和 32 不在同一阵营，31 仍可能攻击 32

## 运行时 API

```lua
-- 判断敌对关系
local is_enemy = player1:is_enemy(player2)

-- 设置敌对关系
player1:set_hostility(player2, true)  -- 设为敌对
player1:set_hostility(player2, false) -- 设为友好

-- 获取敌对玩家组
local enemy_group = player:get_enemy_player_group()
```
