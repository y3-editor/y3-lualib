# Y3 地形系统

地形编辑相关的常量和配置。

## 📋 地形模式 (TerrainMode)

| 模式 | 值 | 说明 |
|------|---|------|
| `Classic` | 0 | 经典地形模式 |
| `Blend` | 1 | 混合地形模式 |

## 📋 笔刷类型 (BrushType)

### 基础笔刷

| 类型 | 值 | 说明 |
|------|---|------|
| `Circle` | 0 | 圆形笔刷 |
| `Square` | 1 | 方形笔刷 |
| `Diamond` | 2 | 菱形笔刷 |
| `Rect` | 3 | 矩形笔刷 |

### 轮廓笔刷

| 类型 | 值 | 说明 |
|------|---|------|
| `CircleContour` | 100 | 圆形轮廓 |
| `ExCircleContour` | 101 | 扩展圆形轮廓 |
| `SquareContour` | 102 | 方形轮廓 |
| `DiamondContour` | 103 | 菱形轮廓 |

---

## 📋 坐标类型 (CoordinateType)

| 类型 | 值 | 格子大小 | 说明 |
|------|---|----------|------|
| `Terrain` | 0 | 1.0 | 地形格子 |
| `Subcoord2x2` | 1 | 0.5 | 2x2 子坐标（植被） |
| `Subcoord4x4` | 2 | 0.25 | 4x4 子坐标（涂鸦） |
| `Foliage` | 1 | 0.5 | 植被坐标 (= Subcoord2x2) |
| `Graffiti` | 2 | 0.25 | 涂鸦坐标 (= Subcoord4x4) |

### 格子大小常量

```python
# 植被格子宽度
FoliageWidth = 0.5
InvFoliageWidth = 2  # 1 / 0.5

# 涂鸦格子宽度
GraffitiWidth = 0.25
InvGraffitiWidth = 4  # 1 / 0.25
```

## 🔧 地形生成工具函数

### 噪声地图生成

```python
from commons.utils.noise_map import gen_noise_map, gen_terrain_map

# 生成基础噪声地图
noise_map = gen_noise_map(
    width=256,
    height=256,
    seed=12345,
    scale=1.0,
    fudge=1.0,
    exp=1.0,
    octaves=5,
    lacunarity=2.0,
    gain=0.5
)

# 生成完整地形数据
terrain_data = gen_terrain_map(
    width=256,
    height=256,
    seed=12345
)
```

### 岛屿渐变地图

```python
from commons.utils.noise_map import gen_island_gradient_map, gen_circle_gradient_map

# 生成岛屿渐变（边缘衰减）
island_gradient = gen_island_gradient_map(width, height, b=5.5, c=0.083)

# 生成圆形渐变
circle_gradient = gen_circle_gradient_map(width, height, r=0.5)
```

## 🔧 Lua 中使用地形

### 获取地形高度

```lua
-- 获取指定位置的地形高度
local height = y3.terrain.get_height(x, z)

-- 获取带插值的高度
local smooth_height = y3.terrain.get_smooth_height(x, z)
```

### 修改地形

```lua
-- 设置地形高度
y3.terrain.set_height(x, z, new_height)

-- 刷新地形区域
y3.terrain.refresh_area(x, z, width, height)
```

## ⚠️ 注意事项

1. **坐标系统**：Y3 使用左下角原点，Y轴向上，X轴向右
2. **格子大小**：默认地形格子为 1.0 单位
3. **子坐标**：植被和涂鸦使用更细的子坐标系统
4. **边界检查**：确保坐标在世界边界内
