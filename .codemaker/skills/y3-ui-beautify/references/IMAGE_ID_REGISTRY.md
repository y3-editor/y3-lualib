# 图片 ID 速查表 (IMAGE_ID_REGISTRY)

> **⚠️ 美化 UI 时必须优先查阅本文件！**
> 
> 本文件整理了**引擎内置/官方通用**的图片 ID。美化 UI 时直接使用这些 ID，避免凭空编造导致显示异常。
> 
> ⚠️ **禁止将项目自定义资源放入本文件！** 自定义资源仅在特定项目中可用，换项目会显示空白。

---

## 📌 使用指南

1. **优先使用本文件中的 ID** — 这些是引擎内置/官方通用资源
2. **按组件类型查找** — 每个分类对应特定场景
3. **无合适 ID 时可省略** — 让引擎使用默认值，用户在编辑器中手动设置
4. **需要自定义资源时** — 让用户在编辑器中手动选择图片，不要硬编码 ID
5. **所有 ID 均来自官方资源库** — 通过 `get_official_editor_icon` 查询验证

## 11. 按组件类型速查

### Image (type=4) → `image` 属性
```json
{"type": 4, "image": 906864, ...}  // 框体资源
{"type": 4, "image": 132046, ...}  // 弹窗背景
```

### Layout (type=7) → `mask_image` 属性（圆形遮罩）
```json
{
  "type": 7,
  "clip_enabled": true,
  "clipping_type": 1,
  "mask_image": <图片ID>,
  ...
}
```

### Button (type=1) → `normal_picture` 属性
```json
{"type": 1, "normal_picture": 906009, ...}  // 按钮(蓝色大按钮hover)
{"type": 1, "normal_picture": 132004, ...}  // 黑色普通按钮(P4风格)
{"type": 1, "normal_picture": 132049, ...}  // 绿色圆形按钮(P4风格)
```

### Progress (进度条) → `image` 属性
```json
{"type": 4, "image": 906858, ...}  // 进度条(boss血条)
{"type": 4, "image": 906001, ...}  // 进度条(玩家血条)
```

### ScrollView (type=10) → `bg_image` 属性
```json
{"type": 10, "bg_image": 999, ...}  // 透明背景
```

## 12. 如何添加新的图片 ID

当发现新的有用图片 ID 时，按以下格式添加：

```markdown
| 用途 | 图片 ID | 名称 | 说明 |
|------|--------|------|------|
| xxx | `123456789` | xxx | xxx |
```

**添加规则：**
- ✅ 引擎内置资源 — 可以添加
- ✅ 官方资源（通过 `get_official_editor_icon` 可查到的） — 可以添加
- ❌ 项目自定义资源（`134xxxxxx` 等查不到的） — **禁止添加**

**验证方法：**
使用 MCP 工具 `get_official_editor_icon` 查询 ID，能返回名称和标签的就是官方资源。
