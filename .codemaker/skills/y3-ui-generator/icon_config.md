# 图片资源配置

本文件定义了 Y3 UI Generator 使用的所有图片资源 ID。包含三大类：
1. **图片预设**（Image Presets）— 用于 `data-preset` 属性
2. **按钮预设**（Button Presets）— 用于按钮的 `data-preset` 属性
3. **自动背景图**（Auto Background）— 根据 layout 尺寸自动选取
4. **默认按钮图片**（Default Button Images）— 无预设时的默认值

> **扩展方法**：添加新资源时，在对应分类的表格中新增一行，同时更新 `scripts/html_to_y3_ui.py` 中对应的字典。

## 1. 图片预设（Image Presets）

在 HTML 中通过 `data-preset="<预设名>"` 引用，也可再用 `data-image` 覆盖。

| 预设名 | 资源描述 | 图片 ID | 使用场景 |
|--------|----------|---------|----------|
| `panel-bg` | 通用面板背景（弹窗底图） | `132046` | 弹窗、面板底图 |
| `frame` | 通用边框（方形图标框） | `132055` | 卡片、槽位边框 |
| `dark-bg` | 深色背景（黑色底板） | `906073` | 遮罩、暗色区域 |
| `slot` | 槽位/格子背景（技能空格） | `906176` | 物品格、技能格 |
| `divider` | 分割线/装饰条 | `906131` | 区域分隔 |
| `shadow` | 阴影/暗色遮罩 | `906550` | 投影效果 |
| `highlight` | 高亮/选中效果 | `906144` | 选中态、激活态 |
| `border-glow` | 发光边框（金色图标框） | `132000` | 稀有度、品质边框 |
| `item-bg` | 物品背景（装备框灰色） | `906852` | 道具、装备底图 |
| `title-bar` | 标题栏背景 | `906111` | 面板顶部标题条 |

## 3. 自动背景图（Auto Background Injection）

转换器自动为缺少背景图的 layout 节点注入默认背景图。图片 ID 按 layout 尺寸选取。

| 资源描述 | 选取规则 | 图片 ID |
|----------|----------|---------|
| 背景图-全屏 | 宽高比 = 1920/1080 (≈1.78) | `999` |
| 背景图-大 | 宽 > 600 **且** 高 > 400 | `132046` |
| 背景图-中 | 800 > 宽 > 200 **且** 800 > 高 > 200 | `906957` |
| 背景图-小 | 宽 < 200 **且** 高 < 200 | `906073` |
| 背景图-超宽 | 宽高比 > 6 | `134092` |
| 背景图-宽条 | 6 > 宽高比 > 3 | `134092` |
| 背景图-横幅 | 3 > 宽高比 > 2 | `100062` |
| 背景图-默认 | 其他尺寸 | `999` |

### 注入判断规则

一个 layout 被认为**已有背景图**（不再注入）当且仅当：
- children 中存在 **image (type=4)** 节点
- 该 image 的 **名称包含** `bg`、`background`、`_bg_` 关键词
- 该 image 的 **面积 ≥ layout 面积的 80%**

### 注入的节点属性
- **名称**: `{layout名}_bg`
- **类型**: image (type=4)
- **尺寸**: 与 layout 完全相同
- **适配**: 四向全开 (adapter=all)
- **九宫格**: 关闭
- **位置**: children[0]（最底层）

### 跳过条件
1. 已有符合条件的 bg image
2. 模板节点（`_is_template`）的内部结构
3. 非 layout 类型的节点
