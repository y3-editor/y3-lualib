# 第一阶段：高度图生成

## 概述
根据真实地理坐标（经纬度）生成上海市的 256x256 地形高度图。

## 输入
- 上海市行政边界 GPS 坐标（26个锚点）
- 黄浦江 GPS 轨迹（18个描点）
- 淀山湖 GPS 坐标
- SRTM 高度数据

## 输出
- `output/shanghai_terrain.csv` - 256x256 高度图数据

## 地形类型
| 类型 | 说明 | 高度范围 |
|------|------|----------|
| cliff | 悬崖/边界 | 0（边界） |
| ground | 地面 | 1.2 - 3.7 |
| water | 水体 | - |

## 水体分类
| 类型 | 深度值 | 说明 |
|------|--------|------|
| water + deep (0.3) | 0.3 | 黄浦江 |
| water + deep (0.4) | 0.4 | 淀山湖 |
| water + deep (0.5) | 0.5 | 东海 |
