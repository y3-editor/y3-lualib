# 第二阶段：纹理图生成

## 概述
根据高度图数据生成对应的**纹理图**，仅记录地形（ground）的纹理信息。

## 纹理图格式
- **地形纹理**：`(<int: 地形纹理ID>, <int: 强度>)`
  - 例如：`(227, 100)` 表示使用纹理ID=227，强度=100
- **非地面（水体/悬崖）**：记录为 `0`，在实际应用时使用第一阶段的数据

## 纹理ID来源
纹理数据来源于Y3引擎的以下位置：

1. **地图纹理列表**: `maps/EntryMap/texture.json`
   - 使用 `pickle.loads(zlib.decompress(...))` 解压
   - 格式：`[227, 229, 226, 230, 232, 228, 231]`

2. **纹理定义**: `H:\uat\src\Server\server\engine\dm\commons\datas\editor_terrain_texture.py`
   - `str_8` = `tex_id`（纹理ID）

3. **纹理名称**: `H:\uat\src\Server\server\engine\dm\commons\datas\editor_export_language_data_zh.py`
   - `str_5` = `name`（名称，编码后映射）

### 纹理分配策略（默认）
根据地面高度自动选择纹理：
- 低地 (0-30): 凹凸泥地、碎砖石地
- 中地 (30-100): 褐碎石地、方块苔砖
- 高地 (100-200): 浅绿草皮、翠绿草皮
- 最高地 (200+): 方整砖地
