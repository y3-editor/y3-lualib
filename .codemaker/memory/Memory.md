### 游戏概述
玩家从5个英雄中选择一个作为主角，控制英雄击杀10波怪物。击杀怪物可获得金币、经验和装备掉落。英雄死亡后5秒自动复活，但死亡次数达到5次则游戏失败。成功击杀全部10波怪物则游戏胜利。
